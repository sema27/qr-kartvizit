# 💼 QR Kartvizit (Premium Digital Business Card)

Modern, hızlı ve etkileyici bir şekilde iletişim bilgilerini paylaşmanı sağlayan **QR tabanlı dijital kartvizit uygulaması**.

Bu proje, klasik kartvizitlerin yerini alan; **şık, minimal ve premium tasarıma sahip** kişisel kartvizit sayfaları oluşturmanı sağlar.

---

## ✨ Özellikler

- 📇 Dijital kartvizit oluşturma
- 🔗 QR kod ile anında paylaşım
- ✏️ Kartvizit düzenleme
- 📱 Mobil uyumlu modern tasarım
- 🎨 Premium UI (minimal + modern)
- 🔐 Firebase Authentication
- ☁️ Firestore Database
- 📦 QR kod indirme

---

## 🚀 Nasıl Çalışır?

1. Kullanıcı kayıt olur / giriş yapar  
2. Kartvizit bilgilerini girer  
3. Sistem otomatik QR kod oluşturur  
4. QR kod paylaşılır  
5. Karşı taraf QR kodu okutarak tüm bilgilere erişir

## ⚙️ Kurulum

Projeyi lokalinde çalıştırmak için:

```bash
git clone https://github.com/sema27/qr-kartvizit.git
cd qr-kartvizit
npm install
npm run dev
```
